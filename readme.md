# Sales Analytics Dashboard

## Project Overview
Developed an interactive Sales Analytics Dashboard as a team project to analyze sales performance, profitability, customer behavior, and regional trends using a supermarket sales dataset. The dashboard enables stakeholders to monitor key business metrics and make data-driven decisions.

## Team Project

This project was completed collaboratively as part of a team effort. Team members contributed to different stages of the analytics workflow, including data preparation, analysis, dashboard development, and documentation.

## My Contributions
I was primarily responsible for:

- Importing and preparing the dataset in Power BI..
- Creating DAX measures and KPI calculations.
- Designing and developing the Power BI dashboard.

## Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Power Query
- CSV Dataset

## Project Workflow
1. Collected and reviewed the supermarket sales dataset.
2. Cleaned and transformed the data.
3. Defined business KPIs.
4. Created DAX measures for analysis.
5. Developed interactive dashboards.
6. Derived actionable business insights.

## Key KPIs
- Total Sales
- Total Profit
- Total Orders
- Average Rating
- Quantity Sold
